# 🟢 Tax DZ - تطبيق المساعد الذكي للمنازعات الضريبية

## نبذة عن التطبيق

تطبيق **Tax DZ** هو منصة ذكية متكاملة تساعد المستخدمين على حل المنازعات الضريبية وديًا أو قضائيًا، وفقًا لأحدث تعديلات القانون الجزائري لسنة 2026.

## المميزات الرئيسية

### 🤖 مساعد ذكي AI
- الإجابة على أسئلة المستخدمين في الوقت الفعلي
- شرح مفصل لجميع الإجراءات القانونية
- تقديم المشورة القانونية المتخصصة

### 📚 قاعدة قوانين شاملة
تشمل جميع الفصول والأقسام:
- **الفصل الأول**: الإجراءات الأولية لدى إدارة الضرائب
  - الآجال والمواعيد
  - شكل ومحتوى الشكاوى
  - التأجيل القانوني للدفع
  - الوكالة والتحقيق
  - اختصاصات المديريات

- **الفصل الثاني**: الإجراءات أمام لجان الطعن
- **الفصل الثالث**: الإجراءات أمام المحكمة الإدارية
- **الفصل الرابع**: إجراءات الطعن أمام مجلس الدولة

### 🛠️ أدوات مساعدة
- **حاسبة الآجال**: للتذكير بالمواعيد النهائية
- **نماذج قانونية**: قوالب جاهزة للشكاوى والطعون
- **استشارات قانونية**: خدمات استشارية متخصصة
- **دورات تدريبية**: برامج تعليمية شاملة

### 🎨 تصميم احترافي
- ألوان أخضر وأزرق (الهوية المصرية والعربية)
- واجهة نظيفة وجذابة
- متوافق مع جميع الأجهزة (Responsive)
- تجربة مستخدم محسّنة

## البدء السريع

### المتطلبات
- Node.js 16+
- npm أو yarn
- MongoDB (اختياري)

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/ayaaya200306-del/startup-academy.git
cd startup-academy/tax-dz

# تثبيت المتطلبات
npm install

# إعداد متغيرات البيئة
cp .env.example .env

# تشغيل في بيئة التطوير
npm run dev
```

### التشغيل

سيعمل التطبيق على `http://localhost:5000`
